# 🛍️ Intelligent Shopping Agent

An AI-powered shopping assistant built using **IBM watsonx Orchestrate**, **IBM Granite Models**, and **LangFlow** to help users make smarter purchasing decisions through personalized product recommendations, intelligent product comparisons, price insights, and shopping guidance.

---

## 📌 Problem Statement

Online shoppers often face challenges such as information overload, fake reviews, changing prices, and difficulty comparing products across multiple e-commerce platforms. Traditional recommendation systems usually provide generic suggestions and fail to consider user preferences, budget, and real-time shopping trends.

The Intelligent Shopping Agent addresses these challenges by acting as an AI-powered shopping companion that delivers personalized recommendations and data-driven buying insights.

## 🧠 Agentic AI Workflow

The solution follows a multi-agent architecture where each AI agent performs a dedicated task.

### Product Knowledge Agent
- Retrieves product information
- Summarizes specifications
- Collects ratings and reviews

### Recommendation Agent
- Suggests products based on:
  - Budget
  - Brand preference
  - Features
  - User requirements

### Price Intelligence Agent
- Analyzes pricing trends
- Suggests best buying time
- Identifies better deals

### Review Analysis Agent
- Summarizes customer reviews
- Detects suspicious review patterns
- Highlights product pros and cons

## ⚙️ LangFlow Components Used

### Chat Input
Accepts user shopping queries and preferences.

### IBM watsonx AI Agent
Processes user requests using IBM Granite models.

### Granite Model
**Granite-4.0-8B-Instruct**

### Chat Output
Displays recommendations, comparisons, and shopping insights.

### File Component
Stores product information, pricing details, and review datasets.

### Vector Store / Embeddings
Enables Retrieval-Augmented Generation (RAG) for efficient product knowledge retrieval.
